# 酸碱中和滴定实验操作手册
## 一、实验目的
1. 掌握酸碱中和滴定的基本原理（利用酸碱反应定量计算未知溶液浓度）。
2. 学会酸式滴定管、碱式滴定管的正确使用方法（查漏、洗涤、润洗、装液、排气泡、读数）。
3. 通过实验计算未知浓度 NaOH 溶液的准确浓度（以已知浓度 HCl 溶液为标准液）。
4. 培养实验操作的规范性和数据记录的严谨性，误差分析能力。

## 二、实验原理
1. 反应方程式：HCl（盐酸，强酸） + NaOH（氢氧化钠，强碱） → NaCl（氯化钠） + H₂O（水），反应计量比为 1:1。
2. 定量关系：c₁V₁ = c₂V₂（c₁为标准液 HCl 浓度，V₁为 HCl 消耗体积；c₂为未知液 NaOH 浓度，V₂为 NaOH 取样体积）。
3. 指示剂选择：选用酚酞试液（变色范围 pH 8.2-10.0），滴定终点现象为“溶液由无色变为浅粉红色，且 30 秒内不褪色”。

## 三、实验器材与试剂
### （一）器材
1. 酸式滴定管（50mL，精度 0.01mL，用于装酸性标准液 HCl）
2. 碱式滴定管（50mL，精度 0.01mL，用于装碱性未知液 NaOH）
3. 锥形瓶（250mL，3 个，用于承接待测液并反应）
4. 移液管（25mL，1 支，准确量取未知浓度 NaOH 溶液）
5. 烧杯（100mL，2 个，分别盛放 HCl 和 NaOH 溶液）
6. 玻璃棒（1 根，辅助转移溶液）
7. 铁架台（带滴定管夹，1 套，固定滴定管）
8. 洗瓶（1 个，装蒸馏水，用于洗涤器材）
9. 滤纸（若干，擦干滴定管外壁水珠）

### （二）试剂
1. 标准盐酸溶液（c(HCl) = 0.1000mol/L，已知浓度，作为滴定剂）
2. 未知浓度氢氧化钠溶液（待测定浓度，约 0.1mol/L）
3. 酚酞试液（0.1% 乙醇溶液，作为指示剂）
4. 蒸馏水（用于洗涤器材、稀释残留液）

## 四、实验步骤（分 4 阶段）
### 阶段 1：器材洗涤与查漏
1. 滴定管洗涤：
   - 先用自来水冲洗酸式、碱式滴定管各 3 次（每次注水约 1/3 容积，转动管壁后倒掉）；
   - 再用蒸馏水洗涤各 3 次（操作同前）；
   - 最后“润洗”：酸式滴定管加 5mL 标准 HCl 溶液，转动管壁润洗内壁后倒掉，重复 3 次；碱式滴定管加 5mL 未知 NaOH 溶液，润洗 3 次（润洗目的：避免残留蒸馏水稀释待装溶液，导致浓度误差）。
2. 查漏：
   - 酸式滴定管：关闭活塞，装水至“0”刻度以上，静置 2 分钟，观察活塞处是否漏水；若漏水，重新涂抹凡士林（将活塞取出，在两端涂薄层凡士林，放回后旋转至透明）。
   - 碱式滴定管：关闭橡胶管中的玻璃球，装水至“0”刻度以上，静置 2 分钟，观察橡胶管与玻璃管连接处是否漏水；若漏水，更换橡胶管或玻璃球。

### 阶段 2：装液与排气泡
1. 酸式滴定管装液：将标准 HCl 溶液倒入酸式滴定管中，液面高于“0”刻度，打开活塞，让溶液缓慢流出，排出管尖气泡（气泡会导致读数偏大），最后调节液面至“0.00mL”或略低于“0”刻度，记录初始读数 V₁（精确到 0.01mL，如 0.23mL）。
2. 碱式滴定管装液：将未知 NaOH 溶液倒入碱式滴定管中，液面高于“0”刻度，弯曲橡胶管使玻璃尖嘴向上，挤压玻璃球，让溶液流出排出气泡，调节液面至“0.00mL”或略低于“0”刻度，记录初始读数 V₃（精确到 0.01mL）。

### 阶段 3：移液与滴定（平行实验 3 次，取平均值）
1. 移取待测液：用 25mL 移液管准确量取 25.00mL 未知 NaOH 溶液（移液管需先润洗 3 次：用少量 NaOH 溶液润洗内壁），放入 250mL 锥形瓶中，滴加 2-3 滴酚酞试液，振荡均匀（此时溶液无色）。
2. 滴定操作：
   - 将锥形瓶放在酸式滴定管下方，调节滴定管高度，使管尖伸入锥形瓶瓶口约 1cm（不接触瓶壁）；
   - 左手控制酸式滴定管活塞（无名指和小指弯曲托住活塞下方，拇指和食指转动活塞），右手持锥形瓶颈部，边滴加 HCl 溶液边顺时针振荡锥形瓶（让溶液充分混合）；
   - 初始阶段：可快速滴加（每秒 3-4 滴），当溶液局部出现粉红色且快速消失时，改为“逐滴滴加”（每滴间隔 1-2 秒）；
   - 滴定终点：当滴入最后 1 滴 HCl 溶液后，锥形瓶内溶液由无色变为浅粉红色，且静置 30 秒内不褪色，立即停止滴定，记录酸式滴定管最终读数 V₂（精确到 0.01mL，如 24.56mL）。
3. 数据记录：计算本次 HCl 消耗体积 V(HCl) = V₂ - V₁（如 24.56mL - 0.23mL = 24.33mL），将数据填入实验记录表。
4. 重复操作：将锥形瓶洗涤干净（用蒸馏水洗涤 3 次，无需润洗），重复步骤 1-3，完成 3 次平行实验，确保 3 次消耗体积的相对偏差≤0.2%（若偏差过大，需重新实验）。

### 阶段 4：实验后处理
1. 器材清洗：
   - 滴定管：倒出剩余溶液（不可倒回原试剂瓶，避免污染），用自来水冲洗 3 次，再用蒸馏水冲洗 3 次，酸式滴定管打开活塞晾干，碱式滴定管取下橡胶管晾干。
   - 锥形瓶、移液管：用自来水冲洗后，蒸馏水润洗，放回原位。
2. 试剂整理：剩余标准 HCl、未知 NaOH 溶液密封保存，酚酞试液盖紧瓶盖，放回试剂柜。
3. 数据计算：根据 c(NaOH) = [c(HCl) × V(HCl)] / V(NaOH)，计算 3 次平行实验的 c(NaOH)，取平均值作为最终结果（如 c(HCl)=0.1000mol/L，V(HCl)平均=24.30mL，V(NaOH)=25.00mL，则 c(NaOH)=0.0972mol/L）。

## 五、注意事项（安全与误差控制）
1. 安全操作：
   - 盐酸、氢氧化钠溶液均有腐蚀性，若溅到皮肤或衣物上，立即用大量自来水冲洗（皮肤接触后可涂稀硼酸溶液中和 NaOH，涂碳酸氢钠溶液中和 HCl）；
   - 滴定管使用时不可俯视或仰视读数，避免摔落损坏（玻璃材质易碎）。
2. 误差控制：
   - 润洗步骤不可省略（未润洗滴定管会导致待装溶液浓度偏低，消耗体积偏大，计算结果偏大）；
   - 读数时视线需与滴定管内液面的凹液面最低处相平（俯视读数会导致读数偏小，仰视读数偏大）；
   - 滴定终点判断需准确（滴加过量 HCl 会导致消耗体积偏大，结果偏大；未达到终点会导致消耗体积偏小，结果偏小）；
   - 平行实验时，每次均需重新调节滴定管液面至“0.00mL”或相近刻度，减少管尖残留误差。
